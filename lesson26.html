Accessing Saved Data

So far, all the data we've used (variables, lists, dictionaries) disappears when the program finishes. If you want to save data permanently, you need to use **files**.

Python can open and read text files (like `.txt` files) that are saved on your computer. The process has three main steps: **Open**, **Read**, and **Close**.

The `with open()` Statement

The safest and easiest way to work with files is using the **`with open()`** statement. It automatically closes the file for you, which is important to prevent errors! 


with open("my_story.txt", "r") as file:
    # 'r' means "read" mode
    
    # 1. Read all the words into one string variable
    story_content = file.read() 
    
    print(story_content)

Key Parts to Remember:

  • **`open("filename.txt", "r")`**: The `open()` function needs the file name and a mode. We use **`"r"`** for **r**eading.
  • **`as file`**: We give the opened file a variable name, often just `file` or `f`.
  • **`.read()`**: This method grabs *all* the content from the file and saves it into a single string variable.

Tip: Once you leave the indented `with` block, the file is safely closed!
