Lesson 7: Making Choices (If/Else) üö¶

The `if` Statement: A Computer's Brain

A program that can only do one thing is boring. The **`if` statement** is the first step to making a smart program! It tells the computer: **"IF this rule is true, THEN do this block of code."**

There are two important parts to remember:

  1. You must end the `if` line with a **colon** (`:`).
  2. The code that runs must be pushed in (**indented**) with a tab or four spaces.

Example 1: A Simple Condition


score = 100

if score > 90:
    print("Great job! You got an A!")
    print("That was a perfect score!")

The `else` Statement: The Backup Plan

What if the rule in the `if` statement is **false**? That's where **`else`** comes in. It's the backup plan!

The computer checks the `if` rule. If it's **True**, it does the `if` code. If it's **False**, it skips the `if` code and jumps straight to the `else` code.

Example 2: Using If and Else Together


is_cold = False

if is_cold == True:
    print("Put on a jacket.")
else:
    print("Wear a t-shirt!")

(Because `is_cold` is `False`, the computer skips the first print and runs the code under `else`.)
