Making Functions Flexible

In the last lesson, our function always did the exact same thing. But real-world functions, like `print()`, can change what they do based on the **ingredients** you give them!

These ingredients are called **Arguments** (or **Parameters**). When you define the function, you give these ingredients temporary names, which are stored inside the parentheses `()`.

Example 1: A Customized Greeting


# 1. DEFINE: The function needs one ingredient called 'name'
def custom_greeting(name):
    print("Hello, " + name + "!")
    print("Welcome back to Python Pals.")

# 2. CALL: We give it the specific ingredient (the argument)
custom_greeting("Leo")
custom_greeting("Sam")

The first time it runs, the `name` variable inside the function is "Leo". The second time, it's "Sam".

Example 2: A Recipe with Two Ingredients

You can pass as many arguments as you need, separated by commas. 


# The function needs 'food' and 'price'
def lunch_order(food, price):
    print("You ordered " + food + ".")
    print("That will be $" + str(price) + ".") # price is a number, so we turn it into a string!

# Give it two ingredients
lunch_order("Pizza", 8.50)
