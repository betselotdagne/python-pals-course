The Importance of Index (Position)

A list is an ordered collection, meaning every item has a specific place. This place is called the **index**. To find a specific item in the list, you need to know its index number.

üö® Remember: Python always starts counting from **0**! The first item is at index 0, the second is at index 1, and so on.

Example 1: Accessing an Item

You access an item by writing the list variable name followed by the index number inside **square brackets `[]`**. 


pals = ["Sam", "Leo", "Mia", "Zoe"]

# "Sam" is at index 0
print(pals[0]) 

# "Mia" is at index 2
print(pals[2])

Example 2: Negative Indexing (Counting from the end)

You can also count backward using negative numbers:

  • `-1` is the **last** item.
  • `-2` is the second-to-last item.

colors = ["Red", "Blue", "Green", "Yellow"]

# Prints "Yellow"
print(colors[-1]) 

# Prints "Green"
print(colors[-2])
