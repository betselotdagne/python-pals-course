Keys and Values

In Lesson 12, we learned that a **List** stores data in order, and you find items using a number (the index, starting at 0).

A **Dictionary** is different. It stores data using **Key-Value Pairs**. Think of it like an **address book** or a **secret decoder ring**:

  • The **Key** is the word or name you look up (e.g., "Sam").
  • The **Value** is the information you get back (e.g., Sam's phone number).

How to Create a Dictionary

You use **curly braces `{}`** to create a dictionary. Inside, you write the **key**, followed by a **colon `:`**, followed by the **value**.

Example 1: An Animal's Data


animal = {
    "species": "Dog", 
    "name": "Buddy",
    "age": 4, 
    "is_friendly": True
}
print(animal)

Accessing Data with Keys

To get a value out of the dictionary, you use the **key** inside square brackets `[]`, just like you would use an index for a list. 


fav_colors = {
    "Leo": "Blue",
    "Mia": "Pink",
    "Sam": "Green"
}

# The key is "Leo", the value is "Blue"
leos_color = fav_colors["Leo"] 

print("Leo's favorite color is:", leos_color)
