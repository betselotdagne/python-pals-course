The Magic Word: `print()`

In Python, the way you make the computer **say anything** is by using a special command called **`print()`**.

To use it, you write `print`, then a set of parentheses `()`, and inside those parentheses, you put the message you want to display, wrapped in **quotation marks** (`""`).

Example 1: Saying Hello


# This tells Python to display the words "Hello, world!"
print("Hello, world!")

Example 2: Printing a Simple Number

You can also print numbers, but you don't need quotation marks for those. 


print(100)
print(5 + 5)
