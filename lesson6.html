Data Types: The Four Main Kinds

In the last few lessons, you've used different kinds of data: numbers and words. Python is smart, but it needs to know what kind of data it's holding to use it correctly. There are four main types you need to know now:

  • **Integer (`int`):** A whole number, like `10`, `50`, or `-3`. No decimal points.
  • **Float (`float`):** A number with a decimal point, like `3.14` or `9.99`.
  • **String (`str`):** Any sequence of characters, like words or sentences, always in quotes (`"Hello"`).
  • **Boolean (`bool`):** A true/false value. Can only be `True` or `False`. (We'll use these a lot in the next lesson!)

Using the `type()` function

To check what type of data is stored in a variable, you use the special command **`type()`**.

Example 1: Checking Types


my_age = 8         # This is an Integer
my_money = 12.50   # This is a Float
my_name = "Leo"    # This is a String
game_over = False  # This is a Boolean

# Use print() to see what type Python thinks it is!
print(type(my_age))
print(type(my_money))
print(type(my_name))
print(type(game_over))

The Input Problem (A common mistake!)

**Remember:** When you use the `input()` command (from Lesson 5), Python **always** treats the answer as a **String**, even if the user types a number! 


user_number = input("Type a number: ") # Let's say user types 7
print(type(user_number))
# Output will be: <class 'str'>, NOT <class 'int'>
