Combining Rules with `and`

Sometimes, you need two or more things to be true before your program does something. The **`and`** keyword lets you check multiple conditions inside a single `if` statement.

For an `if` statement using **`and`** to be true, **EVERY SINGLE condition** you connect must be true. If even one condition is false, the entire rule is false.

Example 1: Getting a Treat

Imagine a rule: You get ice cream **IF** you finish your homework **AND** clean your room. Both must happen! 


homework_done = True
room_clean = False

if homework_done == True and room_clean == True:
    print("Time for ice cream!")
else:
    print("Finish your chores first.")

(In this case, `room_clean` is `False`, so the condition is **False**, and it prints "Finish your chores first.")

Example 2: Checking a Number Range

The `and` operator is perfect for seeing if a number falls **between** two values. 


number = 15

# Is the number greater than 10 AND less than 20? Yes!
if number > 10 and number < 20:
    print("The number is between 10 and 20.")
