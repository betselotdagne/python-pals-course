Difficulty: Medium

The Challenge: Get from 'S' to 'E'!

You will use a **List of Lists** (Nested Lists, Lesson 21) to store a map. The player will move using `w`, `a`, `s`, `d` commands until they reach the **End**.

Key Concepts Used:

  • **Lists of Lists:** To create the 2D map.
  • **Variables:** To track the player's X and Y position.
  • **While Loop:** To keep the game running.
  • **If/Elif Logic:** To check the move, detect walls (`#`), and find the exit (`E`).

# Map Key:
# '#' = Wall (You cannot move here)
# '.' = Path (You can move here)
# 'S' = Start Position
# 'E' = Exit / Finish Line
# '@' = Player

Step 1: Create the Map and Player Position

The map is a list where each item is a row (another list).


maze = [

]

Step 2: Display the Map (Drawing the Maze)

We need a function to draw the maze every time the player moves.


def draw_maze(m, r, c):
    # Temporarily place the '@' symbol at the player's spot
    m[r][c] = '@'
    
    # Print each row, joining the list of characters back into a string
    for row in m:
        print("".join(row))
        
    # Put the original character back so we can check walls correctly next turn
    m[r][c] = '.' 
    
    print("----------------")
