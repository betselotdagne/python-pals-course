Lists are Mutable (Changeable)

The great thing about Python Lists is that you can change them! This means you can update a misspelled item, add a new item to the end, or take an old item out.

1. Changing an Item

If you know the **index** (position) of the item you want to change, you can treat that position like a variable (Lesson 4) and give it a new value. 


fav_colors = ["Red", "Blue", "Green", "Pink"]
print("Before:", fav_colors)

# Change the item at index 3 ("Pink") to "Yellow"
fav_colors[3] = "Yellow" 
print("After:", fav_colors)

2. Adding an Item (`.append()`)

The easiest way to add a new item is to stick it on the very end of the list using the **`.append()`** method. 


shopping_list = ["Milk", "Eggs"]
print("Before:", shopping_list)

# Add "Bread" to the end
shopping_list.append("Bread") 
print("After:", shopping_list)

3. Removing an Item (`.pop()`)

You can remove an item by telling Python the **index** of the item to take out using the **`.pop()`** method. If you don't give it an index, it removes the last item. 


months = ["Jan", "Feb", "Mar", "Apr"]
print("Before:", months)

# Remove the item at index 1 ("Feb")
months.pop(1) 
print("After:", months)
