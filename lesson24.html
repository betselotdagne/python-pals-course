Lists vs. Tuples

A **Tuple** is very similar to a **List** (Lesson 12). It is an ordered collection of data, but it has one major difference:

  • **Lists** use square brackets `[]` and are **mutable** (changeable).
  • **Tuples** use regular parentheses `()` and are **immutable** (**CANNOT** be changed).

Once you create a tuple, you cannot add new items, remove old ones, or change any of the values. They are "locked down"!

How to Create a Tuple

Just like a list, you put items separated by commas, but this time inside `()`.

Example 1: Coordinates (Immutable Data)

Tuples are great for data that should not change, like the coordinates of a location on a map. 


# A tuple of x and y coordinates
coordinates = (10, 50) 
print(coordinates)

Accessing and Looping (Same as Lists!)

You access items in a tuple using the index number in square brackets `[]`‚Äîit starts counting at **0**! You can also use a `for` loop to look at every item. 


days_of_week = ("Mon", "Tue", "Wed", "Thu", "Fri", "Sat", "Sun")

# Access the first day (index 0)
print("Start of the week:", days_of_week[0]) 

# Loop through all items
for day in days_of_week:
    print(day)

üö® Try to change an item: If you run `days_of_week[0] = "Sunday"`, Python will give you an **error**! This is the main purpose of a tuple.
