The Need for Numbered Loops

In the last lesson, we used a `for` loop to repeat code for every item in a list. But what if you just want to repeat code exactly **five times**, and you don't have a list?

The answer is the special function: **`range()`**.

The `range()` Function

The `range()` function creates a sequence of numbers. You use it inside a `for` loop like this:

1. Looping a Fixed Number of Times

If you put one number inside `range()`, Python starts counting from **0** and stops **before** it reaches your number. 


# This loop runs 5 times (i = 0, 1, 2, 3, 4)
for i in range(5):
    print("Lift Off!")

2. Looping with Start and Stop Numbers

You can tell Python exactly where to start and where to stop. range(**start**, **stop**) 


# This loop counts from 1 up to (but not including) 6.
# i will be 1, 2, 3, 4, 5
for number in range(1, 6):
    print(number)

The variable name (`i` or `number`) gets the value of the current count, which you can use inside the loop!
