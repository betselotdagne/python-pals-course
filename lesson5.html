Talking Back to the Computer: The `input()` command

So far, your programs can only talk **to** you using `print()`. Now, it's time to let them listen! The **`input()`** command is how your program asks a question and **waits** for the user (your sons!) to type an answer and press Enter.

The question you want to ask goes inside the parentheses `()`.

Saving the Answer

When the user types an answer, you need a **variable** (Lesson 4) to catch and store that information.

Example 1: Asking for a Name


# 1. The program asks the question inside the input()
# 2. The program waits for the answer
# 3. The answer is saved into the variable 'name'
name = input("What is your name? ") 

# We can then print the name back!
print("Hello, " + name + "! Welcome to Python.")

Example 2: Combining Input and Output

You can ask for multiple pieces of information and use the answers in a fun sentence. 


color = input("What is your favorite color? ")
animal = input("What is your favorite animal? ")

print("A " + color + " " + animal + " sounds amazing!")
