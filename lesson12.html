Storing Collections of Data

So far, a single **variable** (Lesson 4) can only hold one thing, like one name or one number. What if you want to store a whole list of things, like all the planets or all your high scores?

In Python, we use a **List**! A list is an organized collection of items. Think of it like a **toy box** where you can keep many different toys (data) together.

How to Create a List

To create a list, you put all the items inside **square brackets `[]`** and separate them with **commas**.

Example 1: A List of Names (Strings)


pals = ["Sam", "Leo", "Mia", "Zoe"] 
print(pals)

# You can even mix data types!
mixed_bag = ["Apple", 5, True, 3.14]
print(mixed_bag)

Counting Items: The `len()` function

To quickly find out how many items are in your list (how many toys are in the box), you use the **`len()`** function. 


groceries = ["Milk", "Eggs", "Bread"]
total_items = len(groceries)

# This will print 3
print(total_items)
