The `return` Keyword

In the last lesson, our function did the work (like adding numbers) and then printed the result. But what if you want to use that result later in your main program?

The **`return`** keyword is like an instruction to the function to **stop** and **give a value back** to where it was called from. This is how the `len()` function works‚Äîit returns the count, which you can then save in a variable.

Example 1: Returning a Sum

Notice how we use the function call (`add_me(5, 10)`) on the right side of the equals sign, just like any other value! 


# Define a function that RETURNS the sum
def add_me(num1, num2):
    total = num1 + num2
    return total # Send the 'total' value back!

# 1. Call the function and save the returned value in a variable
first_answer = add_me(5, 10) 
print(first_answer) # Output: 15

# 2. Use the returned value in another calculation
final_score = first_answer * 2
print(final_score) # Output: 30

Example 2: Returning a String

You can return any type of data: numbers, strings, Booleans, or even lists! 


def get_message(name):
    # Return a complete string
    return "Nice to meet you, " + name + "!"

my_message = get_message("Zoe")
print(my_message) # Output: Nice to meet you, Zoe!

Anything written after a `return` statement in a function will NOT run, because the function stops instantly!
