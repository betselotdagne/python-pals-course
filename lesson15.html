The Need for Loops

Imagine you have a list of 100 names and you want to print a welcome message for every person. Writing `print()` 100 times would take forever! This is where a **loop** is needed.

A **loop** is a way to tell the computer to repeat a set of instructions many times without you having to write them over and over.

The `for` Loop: Iterating Over Lists

The **`for` loop** is the most common kind of loop. It is perfect for doing something to **every single item** in a List (Lesson 12).

How it works:

for **item** in **list_name**:

  • It goes to the list (`list_name`).
  • It picks out the first item and temporarily calls it **`item`**.
  • It runs the indented code.
  • It picks out the next item, calls it **`item`**, and runs the code again, until the list runs out!

Example 1: Saying Hello to Everyone


names = ["Leo", "Sam", "Zoe"]

for name in names:
    # This line runs 3 times, once for each name!
    print("Hello, " + name + "!")

(The output will be: "Hello, Leo!", "Hello, Sam!", "Hello, Zoe!")
