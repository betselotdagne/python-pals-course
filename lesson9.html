The Comparison Operators

In Lessons 7 and 8, we used rules inside the `if` and `elif` statements, like `score > 90`. These rules use **comparison operators** to ask Python a True/False question. The answer to this question is always a **Boolean** (`True` or `False`, which we learned in Lesson 6).

All the ways to compare two things:

  • **Equal to:** `==` (Two equal signs, e.g., `age == 10`)
  • **Not Equal to:** `!=` (e.g., `name != "Bob"`)
  • **Greater Than:** `>` (e.g., `money > 50`)
  • **Less Than:** `<` (e.g., `health < 10`)
  • **Greater Than or Equal To:** `>=` (e.g., `points >= 100`)
  • **Less Than or Equal To:** `<=` (e.g., `temperature <= 32`)

‚ö†Ô∏è **Important!** Do not confuse the **Comparison Operator** (`==`) with the **Assignment Operator** (`=`). A single equals sign (`=`) is only used to store a value in a variable.

Example 1: Using Comparison Operators with Print


number_of_friends = 5

# Is 5 equal to 5? (True)
print(number_of_friends == 5)

# Is 5 less than 3? (False)
print(number_of_friends < 3)

# Is 5 greater than or equal to 5? (True)
print(number_of_friends >= 5)

# Is "cat" not equal to "dog"? (True)
print("cat" != "dog")
