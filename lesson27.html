Saving Data with Files

The most exciting part of using files is saving new data! To do this, you change the **mode** when you open the file in the `open()` function.

Two Ways to Write: Write (`w`) vs. Append (`a`)

When you open a file for writing, you must choose what you want to do with the content that's already there:

1. Write Mode (`"w"`) - **CAUTION!**

The **`"w"`** mode (for **w**rite) will **DELETE everything** that was in the file and start a brand new, empty file. Use this when you want to replace the old content completely. 


with open("my_note.txt", "w") as file:
    # This line completely overwrites the file!
    file.write("This is the NEW message.")

2. Append Mode (`"a"`) - **The Safer Choice**

The **`"a"`** mode (for **a**ppend) is usually safer. It leaves all the old content alone and puts your new content at the **very end** of the file. 


with open("log.txt", "a") as file:
    # This adds the message to the end of the file.
    file.write("User logged in at 10:00 AM.\n") # \n adds a new line

The `.write()` Method

Once the file is open for writing (using `"w"` or `"a"`), you use the **`.write()`** method to pass in the string (text) you want to save. You must give it a string!
