Lesson 2: Numbers and Math üßÆ

Python is a Super Calculator!

Python can do all the math you need, instantly! You use the same symbols you know for basic math, but you need to learn two new ones.

The Math Symbols

* **Addition (Plus):** `+` (e.g., `5 + 3`) * **Subtraction (Minus):** `-` (e.g., `10 - 4`) * **Multiplication:** `*` (The star, not 'x') (e.g., `2 * 8`) * **Division:** `/` (The forward slash) (e.g., `20 / 5`)

Example 1: Basic Calculations


print(10 + 25)
print(4 * 9)
print(50 / 5)

Example 2: Order Matters!

Just like in school, Python does multiplication and division **before** addition and subtraction. Use parentheses `()` if you want Python to calculate something first. 


# Python does 2 * 3 first, then adds 10 (Result: 16)
print(10 + 2 * 3)

# Python does (10 + 2) first, then multiplies by 3 (Result: 36)
print((10 + 2) * 3)
