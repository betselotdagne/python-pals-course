Dictionaries are Changeable!

Just like **Lists** (Lesson 14), **Dictionaries** are **mutable**, meaning you can change them after they are created. You can update an existing value, add a brand-new key-value pair, or remove a key-value pair entirely.

1. Updating a Value

If you use the bracket notation `[]` with an existing **Key** and assign it a new value, Python replaces the old value with the new one. 


user_data = {
    "name": "Leo", 
    "score": 500, 
    "level": 10
}
print("Before:", user_data)

# Leo gets a higher score!
user_data["score"] = 999 

print("After:", user_data)

2. Adding a New Key-Value Pair

This uses the exact same technique as updating a value, but you use a **new key** that doesn't exist yet! 


# We want to add a high_score key to the dictionary.
user_data["high_score"] = 1500 

print("Added:", user_data)

3. Removing a Key-Value Pair (`del`)

To completely remove a key and its value, you use the **`del`** keyword (short for delete) followed by the dictionary name and the key in brackets. 


my_dictionary = {"a": 1, "b": 2, "c": 3}
print("Before del:", my_dictionary)

# Remove the key "b"
del my_dictionary["b"] 

print("After del:", my_dictionary)
