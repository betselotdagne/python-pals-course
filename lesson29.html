Adding Unpredictability

A program that does the same thing every time can be a bit boring! If you want to make a game, a quiz, or a simulation, you need to introduce **randomness**.

The **`random`** module is a built-in Python superpower (Lesson 28) that lets your code make random choices.

1. Rolling a Virtual Die (`random.randint()`)

The most common tool is **`random.randint(a, b)`**. This function picks a random **whole number** between `a` and `b`, including both the start and end numbers. 


import random

# Roll a 6-sided die. The result will be 1, 2, 3, 4, 5, or 6.
dice_roll = random.randint(1, 6)
print("You rolled a:", dice_roll)

2. Picking a Random Item (`random.choice()`)

If you have a **List** and want to pick one item randomly, you use the **`random.choice()`** function. 


import random
pet_names = ["Buddy", "Whiskers", "Tweety", "Finny"]

# Pick one name randomly from the list
chosen_name = random.choice(pet_names) 
print("Your new pet's name is:", chosen_name)

Remember: You must include `import random` at the top of your code before you can use any of its tools!
