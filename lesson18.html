Writing Your Own `print()`

You've used built-in functions like **`print()`**, **`input()`**, and **`len()`**. Now it's time to create your own!

A **Function** is like a **recipe** for a specific task. You write the recipe once, and then you can call it (or run it) as many times as you want without rewriting all the steps. This saves time and makes your code neat!

Two Steps to Use a Function: Define and Call

  1. **Define (Write the Recipe):** You use the keyword **`def`** (short for define) to write the recipe.
  2. **Call (Run the Recipe):** You use the function's name followed by parentheses `()` to run it.

Example 1: A Simple Greeting Function


# 1. DEFINE the function (Write the recipe)
def say_hello():
    print("Welcome to the Code Kitchen!")
    print("Let's get cooking with Python!")

# 2. CALL the function (Run the recipe)
say_hello()
say_hello() # Run it again!

Remember: The code under `def` must be indented. And just like with variables, the function must be defined **before** you try to call it!
