Lesson 11: One or the Other (Or) üß©

The Power of `or`

In the last lesson, we used **`and`** to mean "Rule A **AND** Rule B must both be true." The **`or`** keyword is the opposite: it means "Rule A **OR** Rule B is true (or both are true)."

For an `if` statement using **`or`** to be true, you only need **at least one** of the connected conditions to be true. The only way it is false is if *all* conditions are false.

Example 1: Getting a Ride

Imagine a rule: You get a ride home **IF** it is raining **OR** it is dark outside. If either one is true, you get the ride! 


is_raining = False
is_dark = True

if is_raining == True or is_dark == True:
    print("No walking today, you get a ride!")
else:
    print("You can walk home.")

(In this case, `is_dark` is `True`, so the condition is **True**, and it prints "No walking today, you get a ride!")

Example 2: Checking for a Specific Name

The `or` operator is useful when a program needs to react to one of several acceptable inputs. 


pet_name = "Buddy"

# Is the pet name "Buddy" OR "Sparky"? Yes, it's "Buddy"!
if pet_name == "Buddy" or pet_name == "Sparky":
    print("That's a popular dog name!")
