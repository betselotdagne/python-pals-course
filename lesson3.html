What is a String?

In Python, any time you want to store **text**‚Äîlike a name, a sentence, or even a single letter‚Äîyou use something called a **String**. Think of a string as a line of characters tied together.

The most important rule for strings is that they must always be wrapped in **quotation marks**. You can use double quotes (`"like this"`) or single quotes (`'like this'`).

Example 1: Making a String


# This is a string!
my_message = "I love Python Pals!"
print(my_message)

# This is also a string, using single quotes
another_word = 'Code'
print(another_word)

Example 2: Adding Strings Together (Concatenation)

You can use the **`+`** symbol to stick two strings together. This is called **concatenation**. 


greeting = "Hi"
name = "Tommy"
# This will print "Hi Tommy"
print(greeting + " " + name)
