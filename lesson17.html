The `while` Loop: Repeating Until False

In the last two lessons, we learned the **`for` loop**, which repeats for a fixed number of items or steps. The **`while` loop** is different: it repeats **as long as a condition is True**.

How it works:

The `while` loop first checks the rule. If the rule is **True**, it runs the indented code. When it reaches the end of the indented code, it goes back and checks the rule **again**.

while **condition is True**:

Example 1: Counting to Three

We use a variable as a **counter** inside the loop. The important thing is to change the counter at the end of the loop so that the condition eventually becomes **False**. 


count = 1

# Check: Is 1 less than or equal to 3? Yes (True)!
while count <= 3: 
    print("Loop number:", count)
    
    # IMPORTANT: Change the condition to stop the loop!
    count = count + 1 # This increases count by 1 each time.

print("Loop finished!")
