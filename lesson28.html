The Python Toolbox

Imagine Python is a basic LEGO set. It has all the blocks you need (like `print`, variables, and loops). But what if you want to build a space station or a robot? You need specialized kits!

A **Module** is a file full of extra Python code (functions, variables, etc.) that someone else wrote. By using the **`import`** keyword, you can bring these specialized tools into your program.

Example 1: The `math` Module

The standard Python toolkit is great, but it doesn't have a special tool for, say, finding the square root of a number. The `math` module does! 


# Bring in the math toolbox
import math

# Use the 'sqrt' (square root) function from the math module
# You must write 'math.' before the function name
result = math.sqrt(25) 
print(result) # Output: 5.0

How to Use an Imported Tool:

When you use a tool from an imported module, you always use this format:
**module_name**.**tool_name**()

Example 2: The `sys` Module

The `sys` module gives you information about the computer system running your code. For instance, the value of `sys.version` is a string telling you which Python version you're using. 


import sys

# sys.version is a variable, not a function, so no () needed!
print("I am running on Python version:")
print(sys.version)

Placing the `import` statements at the very top of your Python file is a good habit!
