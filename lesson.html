What are Variables?

Imagine a **variable** is like a special, labeled **box** where you can store one piece of data, like a number or a secret word. You give the box a name, and Python remembers what's inside!

To create a variable, you write the name, then an equals sign (`=`), then the value you want to store.

Example 1: Storing a Number


# Name the box 'my_age' and put 10 inside
my_age = 10
# Use the box's name in a print statement
print(my_age)

Example 2: Storing a Word (String)

You can also store words! Remember to use quotation marks for text (strings). 


# Name the box 'favorite_food' and put the word inside
favorite_food = "Pizza"
print("My favorite food is:")
print(favorite_food)
