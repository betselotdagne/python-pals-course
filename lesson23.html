Lesson 23: Looping Through Dictionaries üîÑ

Looking at Everything in the Address Book

In Lesson 15, you learned that you can use a **`for` loop** to go through every item in a **List**. You can do the same thing with a **Dictionary**!

A dictionary has two parts for every entry: the **Key** and the **Value**. Python gives you three ways to loop through a dictionary depending on which part you need. 


grades = {
    "Math": 95, 
    "Science": 88, 
    "Art": 92
}

Method 1: Looping Through Keys (The Subject)

If you use a simple `for` loop on a dictionary, Python automatically loops through all the **Keys** (the subject names). 


for subject in grades:
    print("The subject is:", subject)
    # You can get the value inside the loop!
    print("The grade is:", grades[subject])

Method 2: Looping Through Key AND Value (`.items()`)

The best way to loop through a dictionary is by using the **`.items()`** method. It gives you both the Key and the Value at the same time! You just need two variable names in your loop:

for **key**, **value** in **dictionary**.items(): 


for subject, grade in grades.items():
    print(subject + " grade is " + str(grade))
