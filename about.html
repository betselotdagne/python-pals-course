üõ†Ô∏è Deployment Instructions (Free Hosting)

You can host this entire website for free using **GitHub Pages** (the easiest method) or **Netlify**. Since the website uses only HTML, CSS, and JavaScript (Brython runs in the browser), you won't need to pay for any server hosting!

Option 1: Deploy with GitHub Pages (Recommended)

  1. **Create a GitHub Account:** If you don't have one, sign up for a free account at github.com.
  2. **Create a New Repository:** Click the **+** sign in the top right corner and choose **New Repository**. Give it a memorable name (e.g., `python-pals-site`). Set it to **Public**.
  3. **Upload Your Files:**
    • Click the **Add file** button, then **Upload files**.
    • Drag and drop all your files (`index.html`, `lessons.html`, `quizzes.html`, `about.html`, `projects.html`, `style.css`, `editor.js`, and `project_guess_number.html`) into the upload area.
    • Click **Commit changes**.
  4. **Enable GitHub Pages:**
    • Go to the **Settings** tab of your new repository.
    • In the left sidebar, click **Pages**.
    • Under **Build and deployment**, select **Deploy from a branch**.
    • Under **Branch**, select `main` (or `master`) and select the `/(root)` folder. Click **Save**.
  5. **View Your Site:** GitHub will now publish your site. After a few minutes (it can take 5-10 minutes), the URL will appear on the same **Pages** settings page (it will look like `yourusername.github.io/python-pals-site`).
